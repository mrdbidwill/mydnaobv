About This AMS Pilot

This project currently builds county outputs from iNaturalist project Fungi of Alabama- AMS FunDiS Local Project (created by cabracrazy on November 10, 2023).

Each county can publish two files: an observation list PDF (with iNaturalist links) and a county PDF guide (one primary image per observation by default). More images are possible, but increase processing and download time.

Field use note: downloaded PDFs are offline-friendly. iNaturalist links require internet access.

Current inclusion rule: results are built from observations that match both project membership and presence of the DNA Barcode ITS field. We are not offering legal advice, and observers may have valid reasons to restrict image licenses. Restricted or missing-license images cannot be included in county guide PDF pages. The observation list is still created from matching records, but image-based county PDF content may be unavailable for those observations.

If you would like to create a project for another state or another iNaturalist project, please email your request using MRDBID's Contact Page at https://mrdbid.com/contact.

Future options under consideration

Questions under active review include whether this should continue to require both project membership and DNA field presence, how to handle the timing gap before sequence data is added, and whether member workflow guidance should prioritize early project membership.

Related data-source question: whether iNaturalist should remain primary, or be complemented by other available sources.
