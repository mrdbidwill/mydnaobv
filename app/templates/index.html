PDF Creation from iNaturalist Observations Project

This project currently builds county outputs from iNaturalist project Fungi of Alabama – AMS FunDiS Local Project (created by cabracrazy on November 10, 2023).

Two files are created for each county with a completed build: an observation list PDF (with iNaturalist links) and a county PDF guide (currently limited to eight images per observation by default). If no exportable county-guide images are available, a placeholder county guide is generated.

Field use note: downloaded PDFs are offline-friendly. iNaturalist links require internet access.

Current inclusion rule: results are built from observations that match both project membership and presence of the DNA Barcode ITS field. Observers may have valid reasons to restrict image licenses. We are not offering legal advice, however, restricted or missing-license images cannot be included in county guide PDF pages. The observation list is still created from matching records, but images may be unavailable for those observations.
