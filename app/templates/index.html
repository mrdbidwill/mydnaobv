How to Read This Data

Each row represents a county or project product built from iNaturalist observations that include the DNA Barcode ITS field. County results are limited to the county/state scope and are de-duplicated by iNaturalist observation ID.

Use the Observation PDF to review index-level records and source links. Use the County PDF/ZIP for printable pages and image exports. If image export is unavailable for an observation, the county output still preserves index parity with a placeholder page and iNaturalist link.

Taxon labels may change as DNA-informed identifications evolve. For critical decisions, review the linked iNaturalist record, comments, and supporting sequence context.

PDF Creation from iNaturalist Observation Projects

County outputs now query all four AMS sequencing projects, then de-duplicate by iNaturalist observation.id so each observation is counted once per county run.

Two files are created for each county with a completed build: an observation list PDF (with iNaturalist links) and a county PDF guide (currently limited to eight images per observation by default). If no exportable county-guide images are available, a placeholder county guide is generated.

Field use note: downloaded PDFs are offline-friendly. iNaturalist links require internet access.

Current inclusion rule: results are built from observations that match both project membership and presence of the DNA Barcode ITS field. Observers may have valid reasons to restrict image licenses. We are not offering legal advice, however, restricted or missing-license images cannot be included in county guide PDF pages. The observation list is still created from matching records, but images may be unavailable for those observations.

As to the accuracy of the genus or species listed in the headings for each observation you should look at all comments and attached DNA records to follow the data trail. This is a relatively new feature that is being used here to help with accurately identifying fungi and is still evolving. Reviewing this data for accuracy is beneficial to everyone involved.
